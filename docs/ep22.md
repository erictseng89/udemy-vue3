## Adding parameters in methods
Methods are funtions, and hence can take parameters. 

```javascript
methods: {
  addCounter(value) {
    this.counter = this.counter + value;
  },
  removeCounter(value) {
    this.counter = this.counter - value;
  }
}
```
```html
<button v-on:click="addCounter(3)">Add</button>
<button v-on:click="removeCounter(4)">Remove</button>
```
